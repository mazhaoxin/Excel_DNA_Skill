# README

# 项目介绍

这是一个测试skill的项目，该skill可以根据用户需求生成Excel-DNA项目。

# 项目需求

生成一个基于Excel-DNA的Excel插件，该插件提供Ribbon，上面有

- 一个年份选择器，可以选择年份；
- 一个单选框，用来选择星期以周一、周六、周日开头；
- 一个按钮，点击后在当前Excel文件中生成一个Sheet，Sheet名称是年份，Sheet中显示日历，日历的格式为：
    - 月份按照4x3的方式排列，中间空一行/列；
    - 每个月包括title（月份名）、星期和日期，月份名、星期的背景色要与日期不同，以作区分；
    - 每行7天，开头的星期根据上面提到的单选框；
    - 周日和周六用颜色区分；
    - 适当调整列宽、行高和边框线型，使日历显示完整且美观；

为了提高脚本性能，在修改sheet之前请禁用screen update，并在完成后恢复。同时为了防止插件崩溃后screen update仍然被禁用，需要做异常处理。
